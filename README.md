# Areas_BryanDuran

Este proyecto tiene como finalidad realizar operaciones matematicas en cuanto al calculo de areas de figuras.
Fue desarrollado con el lenguaje de programación Java.
El proyecto consta de un menú basico, en donde se le solicita al usuario seleccionar una de las 4 figuras disponibles(cuadrado,triangulo,rectangulo,circulo).
En breve, se le solicitan las medidas en base a la figura seleccionada.
posteriormente se calcula de forma automatica el area de dicha figura.
